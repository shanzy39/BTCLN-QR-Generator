# Changelog

v1.0: Initial release. Home page with Choose Wallet, Quick QR, and About options. Multi-wallet support for Speed, Strike, and Wallet of Satoshi. On-device username editor. Persistent storage on SD card (usernames and last-used wallet). Quick QR shows the QR for the last-used wallet in one tap. Equal-sized QR rendering. Haptic feedback. NFC tag file export with selectable tag type (NTAG213, NTAG215, NTAG216) using the lightning: URI scheme. Generated NFC files include correct BCC bytes, chip-specific Mifare version, and default factory config pages so the stock Flipper NFC app recognizes them as writable. Lightning bolt app icon.
